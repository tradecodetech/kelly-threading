How to print these cards:
  • Press Ctrl+P (Windows) to open print dialog
  • Set paper size to Letter (8.5 x 11)
  • Set margins to Minimum or None
  • Enable Background graphics so colors print correctly
  • Print on cardstock for best results (80–100 lb recommended)
  • Cut along card borders — each card is 3.5 × 5 inches
